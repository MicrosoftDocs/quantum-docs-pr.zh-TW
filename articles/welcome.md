# <a name="get-started-with-the-quantum-development-kit-qdk"></a>開始使用 Quantum Development Kit (QDK)

歡迎使用 Microsoft Quantum Development Kit！  在這裡，您會找到所有必要工具，以學習如何使用 Q# 進行量子程式設計。  為了引導您學習如何使用 Q# 進行量子運算，我們提供了這份「使用者入門指南」，其中針對想要開始編寫量子程式的使用者，以及尚未準備好開始編寫程式碼、但想要深入了解 Q# 和量子程式設計的人員，提供適用的旅程指南。

## <a name="questions-about-quantum-computing"></a>量子運算相關問題

一開始會有一系列的簡短文章，摘要說明「關於量子運算的問題」  。 這些簡短文章可讓新的學習者快速地大致了解量子運算。

[什麼是量子運算？](xref:microsoft.quantum.overview.what)

## <a name="how-to-learn-quantum-computing-with-q"></a>如何使用 Q# 學習量子運算？

Q# 和 Quantum Development Kit 可提供必要工具讓您開發量子運算程式，以解決我們在運算方面所遭遇的最大難題。

針對第五個問題：「如何使用 Q# 學習量子運算？」  ，我們從提供下列概觀開始：

* 說明如何建立[量子亂數產生器](xref:microsoft.quantum.quickstarts.qrng)的使用者入門指南，引導您安裝 QDK 並撰寫第一個 Q# 程式，以及量子概念的簡介。 亂數產生器是「Q# Hello World」樣式應用程式，讓您在幾分鐘內就能建立並執行量子應用程式。

* [搭配 Q# 使用的量子基本知識](xref:microsoft.quantum.write-program)使用者入門指南會引導您撰寫一個 Q# 程式來示範一些關於量子程式設計的基本概念。 如果您尚未準備好開始撰寫程式碼，仍可在不安裝 QDK 的情況下繼續，同時大致了解 Q# 程式設計語言和量子運算的首要概念。

* [格羅弗搜尋](xref:microsoft.quantum.quickstarts.search)使用者入門指南提供 Q# 的範例，摘要示範低階的量子操作，讓您了解 Q# 在表示量子演算法方面的能力。  格羅弗搜尋搜尋快速入門會引導您使用各種程式設計環境 (具有 Python 主機或 .NET 語言主機，以及具有 Visual Studio 或 Visual Studio Code) 來開發程式。

* [影片](https://www.microsoft.com/videoplayer/embed/RE2JOJf)，可深入瞭解如何使用 Q# 程式庫來建立量子程式。  QDK 隨附 Q# 標準和數值程式庫，可提供強大的運算抽象概念，協助您開發這些解決方案。 我們也提供了量子化學程式庫，這個 Q# 程式庫可用於建立量子層級的化學作用模型。

## <a name="next-steps"></a>後續步驟

上述使用者入門指南和文章的目的，是要透過簡介量子運算和 Q# 來引導程式設計者和非程式設計者。  若要繼續學習如何使用 Q# 來編寫量子電腦程式：

* [如何使用 Q # 來學習量子運算](xref:microsoft.quantum.overview.learn)提供使用 Q# 學習量子運算概念的指引。

* 利用我們的[自學型 Quantum Katas](https://aka.ms/try-quantum-katas) 來學習量子程式設計。 這一系列的簡短訓練練習將指導您如何以 Q# 編程和學習量子運算。

* 請探索我們的 [Q# 範例](https://docs.microsoft.com/samples/browse/?languages=qsharp)，並找出您可以使用量子運算解決的問題。

* 在本機電腦上，使用您所選的主機程式設計語言和開發環境進行開發：
  * 使用 [Python](xref:microsoft.quantum.install#develop-with-python) 來開發
  * 使用 [Visual Studio](xref:microsoft.quantum.install#develop-with-c-on-windows-using-visual-studio) 來開發
  * 使用 [Visual Studio Code](xref:microsoft.quantum.install#develop-with-c-using-visual-studio-code) 來開發
  * 使用 [Jupyter Notebook](xref:microsoft.quantum.install#develop-with-jupyter-notebooks) 來開發

* 藉由參與 [QDK](xref:microsoft.quantum.contributing) 開放原始碼專案來成為 Microsoft Quantum 社群的一分子。
