# <a name="more-quantum-computing-learning-resources"></a>更多量子計算學習資源

雖然量子運算 [概念](xref:microsoft.quantum.concepts.intro) 一節中所討論的工具對於量子軟體的任何開發人員而言是基本的，但它們並不是跨越量子電腦程式設計和演算法設計已知的深度或廣度。  由於量子運算是快速開發的欄位，因此沒有任何一項資源擁有學習如何充分利用這些工具來解決問題所需的所有資訊。  基於這個理由，我們編譯了一份參考清單，可能會想要深入瞭解量子電腦程式設計的藝術和美。
本節包含有關量子運算主題深入探討的選取參考。
