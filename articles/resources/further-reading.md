# <a name="more-quantum-computing-learning-resources"></a>更多量子運算學習資源

雖然「配量[計算概念](xref:microsoft.quantum.concepts.intro)」一節中所討論的工具是任何量子軟體發展人員的基礎，但它們不會涵蓋有關量子電腦程式設計和演算法設計已知的深度或廣度。  因為量子運算仍然是快速開發欄位，所以沒有任何一項資源具有學習如何最佳使用這些工具來解決問題所需的所有資訊。  基於這個理由，我們編譯了一份參考清單，可能會想要深入瞭解有關量子電腦程式設計的藝術和優點的讀者。
