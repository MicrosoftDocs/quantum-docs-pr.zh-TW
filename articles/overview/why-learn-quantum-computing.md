# <a name="why-learn-quantum-computing"></a>為什麼要學習量子運算？

量子電腦很棒，但為何要學習量子運算？ 在本節中，我們將說明您現在應學習量子運算的幾個原因。

## <a name="join-the-quantum-revolution"></a>參與量子革命

研究人員正開始勾勒未來量子技術的可能性。 我們正在見證部分人士稱之為第二次量子革命的發軔。 第一次革命的內涵，包括使用量子物理推動全人類的革新性發明，例如雷射和電晶體。 過去十年來，科學開始能夠在個別粒子的層級上掌控量子系統。 其中蘊含了各種可能性，且大部分仍有待探索。 別再觀望了，投身於其中吧！ 您可以開始在形塑未來世界的科技領域貢獻己力。

## <a name="solving-great-challenges-requires-diverse-communities"></a>要克服艱難的挑戰，有賴各類社群的合作

要達到有效率的量子運算，可說是當前最大的挑戰之一。 其複雜性在於有許多不同的層面有待解決：可靠且可擴充的量子硬體、有效率的量子軟體、穩定且低成本的錯誤修正通訊協定、實際的應用程式等等。 我們期待能組成科學家、工程師和開發人員的龐大社群，而共同解決這些深具挑戰的問題。 要形成此社群，必須要有來自各種領域的廣大人際網路。 無論您的背景為何，都可以立即參與量子革命。 歡迎大家加入量子革命的行列！

## <a name="get-inspired-by-the-quantum-world"></a>量子世界的啟發

雖然我們尚未完成可容錯的量子電腦，但我們已觀察到量子運算在傳統電腦科學領域確實有顯著的影響。 過去幾年來，許多新演算法的開發都是以量子運算使用的技術為基礎。 這些就是所謂的量子啟發演算法，可以在傳統電腦上執行。 這些演算法的一些範例包括建議系統，或執行速度比任何舊有傳統方法都快的基因演算法。 藉由學習量子運算，您對運算也會有新的思維，進而改變您對於傳統程式設計的觀點！

## <a name="quantum-computing-is-fun"></a>量子運算很有趣！

除了以上討論的實際影響外，我們也認為量子運算饒富趣味。 學習量子運算是一種有益的體驗，您將習得最新的知識和技能，以利建置未來的技術。
