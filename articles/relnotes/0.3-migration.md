<span data-ttu-id="2447f-140">下列程式碼說明一些新功能：</span><span class="sxs-lookup"><span data-stu-id="2447f-140">The following code illustrates some of the new features:</span></span>

<span data-ttu-id="2447f-144">和中的配置 `using` `borrowing` 是針對單一 qubits、qubit 陣列，以及其中的嵌套元組而新增的：</span><span class="sxs-lookup"><span data-stu-id="2447f-144">Allocations within `using` and `borrowing` are newly possible for single qubits, qubit arrays, and nested tuples thereof:</span></span>
