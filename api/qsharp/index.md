# <a name="q-libraries"></a>Q# 程式庫 #

Q # 標準程式庫包含用來開發量子程式的作業、函式和使用者定義類型。 此外，也包含了以標準程式庫為基礎的其他三個程式庫，包括化學、數值和機器學習程式庫。

如需 Q# 程式庫的詳細資訊，請參閱 <xref:microsoft.quantum.libraries>。
