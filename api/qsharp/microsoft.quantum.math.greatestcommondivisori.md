計算 $a $ 和 $b $ 的最大公因數。 GCD 一律為正數。

```qsharp
function GreatestCommonDivisorI (a : Int, b : Int) : Int
```

### <a name="b--int"></a>b： [Int](xref:microsoft.quantum.lang-ref.int)

計算最大最大公除數的第二個數字



## <a name="output--int"></a>輸出： [Int](xref:microsoft.quantum.lang-ref.int)

$a $ 和 $b $ 的最大公除數
