# <a name="minusc-function"></a>MinusC 函式

命名空間： [Microsoft 量子. 數學](xref:Microsoft.Quantum.Math)

封裝： [Microsoft 量子. 標準](https://nuget.org/packages/Microsoft.Quantum.Standard)


傳回兩個輸入之間的差異。

```qsharp
function MinusC (a : Microsoft.Quantum.Math.Complex, b : Microsoft.Quantum.Math.Complex) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>輸入

### <a name="a--complex"></a>a： [複雜](xref:Microsoft.Quantum.Math.Complex)

要減去的第一個輸入 $a $。


### <a name="b--complex"></a>b： [複雜](xref:Microsoft.Quantum.Math.Complex)

第二個輸入 $b $ 來減去。



## <a name="output--complex"></a>輸出： [複雜](xref:Microsoft.Quantum.Math.Complex)

差異 $a-b $。
