# <a name="israngeempty-function"></a>IsRangeEmpty 函式

只有當輸入範圍是空的時，才會傳回 true。

```qsharp
function IsRangeEmpty (rng : Range) : Bool
```


## <a name="input"></a>輸入

### <a name="rng--range"></a>rng： [範圍](xref:microsoft.quantum.lang-ref.range)
