# <a name="measurefxp-operation"></a>MeasureFxP 操作

命名空間： [Microsoft 量子](xref:Microsoft.Quantum.Arithmetic)

封裝： [Microsoft 量子. 數值](https://nuget.org/packages/Microsoft.Quantum.Numerics)


測量固定點數位、將其值以 Double 傳回，然後將所有的暫存器重設為零。

```qsharp
operation MeasureFxP (fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Double
```


## <a name="input"></a>輸入

### <a name="fp--fixedpoint"></a>fp： [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

要測量的固定點數位。



## <a name="output--double"></a>輸出： [雙精度浮點數](xref:microsoft.quantum.lang-ref.double)
