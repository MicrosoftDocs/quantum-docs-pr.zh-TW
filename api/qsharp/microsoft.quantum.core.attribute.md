# <a name="attribute-user-defined-type"></a>屬性使用者定義型別

命名空間： [Microsoft.. 核心](xref:Microsoft.Quantum.Core)

包： [](https://nuget.org/packages/)


編譯器可辨識的屬性，可用來將使用者自訂類型標示為屬性。

@ Microsoft.Quantum.Core.Attribute()
